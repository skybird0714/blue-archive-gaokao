# blue-archive-gaokao
普通高等学校招生 全国统一考试 蔚蓝档案

试卷排版严格按照了高考数学的排版格式，包括字体字号，行间距等等，少量参考了语文和英语的排版，主要用的word+mathtype

一共38题，单选25题，多选5题，填空5题，大题3题

既有分析题也有计算题，最后的黎曼泽塔函数应该没人做得出来（）

除最后一题，整体试卷偏简单，与游戏关系密切
